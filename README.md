# AB.DESIGN

Modern, professional website for AB.DESIGN - a creative advertising and content creation company.

## Features

- **Responsive Design**: Fully mobile-friendly layout
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Color Scheme**: Navy blue (#2c3e7d) and golden yellow (#fdb913)
- **Sections**:
  - Hero section with eye-catching headline
  - Services showcase (Brand Strategy, Content Creation, Digital Advertising, Creative Design, Video Production, Social Media)
  - Responsive navigation
  - Professional footer

## Technologies

- HTML5
- CSS3 (with Flexbox and Grid)
- Responsive design with media queries

## Quick Start

Simply open `index.html` in your web browser to view the website.

## Customization

To customize the website:

1. **Colors**: Edit the color variables in the CSS section
2. **Content**: Update the text in the HTML sections
3. **Services**: Modify or add service cards in the services section
4. **Images**: Replace placeholder elements with actual images
